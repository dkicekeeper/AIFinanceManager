### 2. Несовместимость валют при переводах между счетами

**Локация:** `AccountActionView.swift:188-197`

**Проблема:**
- При переводе между счетами с разными валютами конвертация выполняется только в валюту **источника** (`account.currency`)
- Но сумма на **получателе** (`targetAccount`) добавляется **без конвертации**
- Это приводит к **некорректным балансам**

**Пример:**
- Счет 1: USD, баланс $100
- Счет 2: KZT, баланс 0 ₸
- Перевод $10 с Счета 1 на Счет 2
- **Ожидаемо:** Счет 1: $90, Счет 2: ~4500 ₸ (при курсе 1 USD = 450 KZT)
- **Фактически:** Счет 1: $90, Счет 2: 10 ₸ ❌

**Текущий код:**
```swift
// AccountActionView.swift:188-197
var convertedAmount: Double? = nil
if selectedCurrency != account.currency {
    convertedAmount = await CurrencyConverter.convert(
        amount: amount,
        from: selectedCurrency,
        to: account.currency
    )
}
// ❌ Не конвертируется для targetAccount!
```

**Рекомендация:**
```swift
// Нужно конвертировать для обоих счетов
let targetAccount = accounts.first { $0.id == targetAccountId }
if let targetAccount = targetAccount,
   selectedCurrency != targetAccount.currency {
    // Конвертировать в валюту получателя
    let targetConverted = await CurrencyConverter.convert(
        amount: amount,
        from: selectedCurrency,
        to: targetAccount.currency
    )
    // Сохранить в транзакции для targetAccount
}
```

### 3. Неправильная обработка валют в `recalculateAccountBalances()`

**Локация:** `TransactionsViewModel.swift:1179-1186`

**Проблема:**
```swift
case .internalTransfer:
    if let sourceId = tx.accountId {
        balanceChanges[sourceId, default: 0] -= amountToUse
    }
    if let targetId = tx.targetAccountId {
        balanceChanges[targetId, default: 0] += amountToUse
    }
```

❌ Использует `amountToUse` (convertedAmount или amount) для **обоих счетов**, но:
- Для источника (sourceId) нужно использовать сумму в валюте источника
- Для получателя (targetId) нужно использовать сумму в валюте получателя
- Сейчас используется одна и та же сумма для обоих

**Пример:**
- Перевод $10 (USD) → счет в KZT (при курсе 450 KZT/USD)
- Source account (USD): -$10 ✅
- Target account (KZT): +10 ₸ ❌ (должно быть +4500 ₸)

**Рекомендация:**
Нужно хранить отдельные суммы для source и target счетов, или хранить информацию о валютах и конвертировать на лету.
