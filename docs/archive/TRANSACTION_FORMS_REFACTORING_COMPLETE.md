## 🎨 Визуальные улучшения

### 1. Ввод суммы

**До:**
```
┌─────────────────────┐
│ Сумма               │
│ [0.00        ▼]     │  ← Маленький TextField + MenuPicker
└─────────────────────┘
```

**После:**
```
┌─────────────────────┐
│ Сумма               │
│                     │
│    1,234.56        │  ← Большие цифры (56pt)
│                     │
│  [₸] [₽] [€] [$]   │  ← FilterChip для валют
└─────────────────────┘
```

### 2. Выбор категории

**До:**
```
┌─────────────────────┐
│ Категория           │
│ [Выберите ▼]        │  ← Стандартный Picker
└─────────────────────┘
```

**После:**
```
┌─────────────────────┐
│ Категория           │
│ [🍔] [🚗] [🏠] [🍕] │  ← Сетка CategoryChip
│ Food  Auto Home Pizza│
└─────────────────────┘
```

### 3. Выбор валюты

**До:**
```
[0.00        ▼]  ← MenuPicker (80pt)
```

**После:**
```
[₸] [₽] [€] [$] [£]  ← FilterChip (все видны сразу)
 ↑ selected
```
